## Getting started

Checking out and installing dependancies
```bash
git clone git@github.com:Cyberbyte-Studios/armabyte.git
composer install
yarn
```

To get a local development copy just run these commands
```bash
php artisan serve
php artisan migrate
php artisan passport:install
```

Compiling CSS files
```bash
gulp --watch
```
